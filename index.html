Rife ESRGAN APP Linux QT Is a redesign (and hopefully an upgrade) from the original Rife ESRGAN App Linux. This program provides easy access to frame interpolation and upscaling on Linux, similar to Flowframes or enhancr on windows.

Features:

  • Discord RPC support for Discord system package and Discord flatpak.
  • Queue up files to render after current render is finished.
  • Scene change detection to preserve sharp transitions.
  • Preview that shows latest frame that has been rendered.
